# DMP Frontend Testing Project

Responzivní statická webová stránka vytvořená podle návrhu ve Figmě.

## Figma

[Figma návrh](https://www.figma.com/design/TumeHzNdMGPfl9r1ImHLSv/dmpagency-frontend-technical-task?node-id=6-67&t=pg1d8pN9vtofgc34-0)

## Použité technologie

- HTML5
- CSS3 (mobile-first, CSS variables)
- Vanilla JavaScript (mobilní menu)

## Spuštění projektu

Otevři `index.html` v prohlížeči, nebo spusť lokální server (např. Live Server ve VS Code / Cursor).

Live URL: [https://tobiaslicek.github.io/DMP-Frontend-Testing-Project/](https://tobiaslicek.github.io/DMP-Frontend-Testing-Project/)

## Struktura projektu

- `index.html` - struktura stránky
- `css/` - styly po komponentech
- `js/main.js` - logika mobilniho menu
- `assets/images/` - obrázky a SVG

## Poznámky

- Třídy jsou psané v BEM konvenci.
- Layout je navržen jako plně responzivní pro mobil, tablet i desktop.
